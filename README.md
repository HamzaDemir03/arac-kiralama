# Araç Kiralama Sistemi

Bu proje, modern bir araç kiralama sisteminin tüm bileşenlerini içerir.

## Proje Yapısı

- `backend/`: REST API ve veritabanı işlemleri
- `web/`: Web arayüzü uygulaması
- `mobile/`: iOS ve Android mobil uygulaması
